SETimes.HR treebank (Croatian)
http://nlp.ffzg.hr/resources/corpora/setimes-hr/

License: CC BY-SA 3.0

The original data was not split to training and test. We take the first 90% for training and the rest for testing.

original size = 7995 sentences, 178981 tokens
training size = 2241 sentences,  50853 tokens
test size     =  249 sentences,   5571 tokens

The corpus is available in the CoNLL-X (CoNLL 2006) file format.
Morphosyntactic tags use the Multext-East 4.0 tagset.
For morphological tagset description: http://nlp.ffzg.hr/data/tagging/msd-hr.html

Paper to read and cite: http://www.ling.uni-potsdam.de/~agic/publications/agic2014-setimes.pdf
