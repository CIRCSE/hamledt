Basque Dependency Treebank (BDT II)

Dan promised to let Koldo know about any interesting results.

This is a newer extended version of the treebank used in CoNLL 2007 shared task.
We are using their training + development data for training, and their test data for testing.

training size = 10104 sentences, 137309 tokens
test size     =  1122 sentences,  14295 tokens

See also the documentation in /net/data/conll/2007/eu/doc.
